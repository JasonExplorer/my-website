# 网站介绍

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

### 开始

```bash
git clone git@github.com:JasonExplorer/my-website.git
cd my-website
npm install
npm run start

```

### 代码框架
```bash
├── blog                           # 博客
├── docs                           # 文档/笔记
├── src
│   ├── components                 # 组件
│   ├── css                        # 自定义CSS
│   ├── pages                      # 自定义页面
├── static                         # 静态资源文件
│   └── img                        # 静态图片
├── docusaurus.config.ts           # 站点的配置信息
├── sidebars.ts                    # 文档的侧边栏
├── package.json
├── tsconfig.json
└── pnpm-lock.yaml
```
